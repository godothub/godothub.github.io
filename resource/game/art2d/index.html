警告

由群友汇总整理，错漏之处欢迎反馈！

https://itch.io/game-assets

itch.io 提供了由全球开发者分享的大量游戏素材，涵盖像素美术、3D 模型、音效与音乐等。无论是独立开发还是原型制作，都能在这里找到灵感和资源。

https://craftpix.net/freebies

Craftpix 收录了许多免费的 2D 游戏资源，包括角色、场景、UI 元素和图标，适合横版动作、塔防、RPG 等多种类型游戏。

https://kenney.nl/assets

Kenney 以高质量、开放授权的免费游戏素材闻名，资源种类丰富，从 2D 图形到 3D 模型、音效一应俱全，特别适合快速原型开发。

https://opengameart.org

OpenGameArt 是一个社区驱动的游戏素材库，用户可以上传和分享免费的艺术、音乐与音效资源，是独立开发者的宝库。

https://www.supergameasset.com/free-game-assets

Super Game Asset 提供了部分免费的高质量 RPG 风格资源，包括角色、装备、场景和 UI 组件，适合追求精美画风的开发者。

https://gameartpartners.com

Game Art Partners 专注于 2D 游戏素材，提供角色、动画和完整套件，适合制作商业级别的独立游戏。

https://www.gamedevmarket.net

GameDev Market 是一个创作者市场，开发者可以购买或出售美术和音频资源，涵盖多种风格与类型，支持独立团队与小型工作室。

http://dotown.maeda-design-room.net

DOTOWN 提供大量免费的像素风图标和小物件，风格统一简洁，特别适合复古或轻量型游戏。

https://www.jusotu.com/category/game.htm

聚搜图 (Jusotu) 是中文资源分享站点，涵盖 2D/3D 素材、UI 元素与场景，部分资源可免费下载，适合国内开发者快速获取资源。

https://www.gameart2d.com

GameArt2D 提供完整的 2D 资源包，包括角色、背景、特效和 UI，风格多样，适合横版与休闲类游戏开发。
